Overview

In this project, we were assgined a client "Skills Label" and we did Usability evaluatiuon for the Registration/Login Page and the Interactive Label Page of the Skills Label website. The whole process was divided into 3 sections:

  • Heuristic Evaluation
  • Test Plan
  • Usability Evaluation

(Work in progress)

REFLECTION

This project gave me an insight about how things work in the professional environment. Setting up meeting times with the client, making agendas, reports, accommodating changes, understanding the client requirements and analyzing whether we can accomplish it within the assigned time. This was a very good learning experience and I understood that Usability Testing Evaluation is much more than asking any random person to use the app.
