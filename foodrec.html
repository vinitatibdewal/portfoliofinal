Food Recognition & Calorie Content Measurement

My Role Research, Solution Design, Android App Development
Deliverables Research Paper, Basic Mobile Application
Timeline September 2016 - April 2017

I wanted to implement machine learning to improve user experience. So, I started a project on “Food Recognition and Calorie Content Measurement”. I did a survey on research done in this field and came to a conclusion that either the datasets used were very small or the user had to enter quantity of food. So, I tried to reduce user efforts by using volume estimation algorithms to calculate the amount of food intake. We worked in a group of four and developed an Android application where the user has to click two pictures, a side view and a top view which gave us data for calculations. We used dataset FOOD101 for training and inception model in Tensorflow framework which gave us an accuracy of 84%. For improving accuracy percentage, I made some changes in the application and gave top five suggestions generated by machine to the user so that they can select the best option and the machine can be trained simultaneously. After selecting correct food item, the calorie content was displayed. I also created a database of other nutrients like carbohydrates, proteins and fats and a database of recipes which was displayed with the calorie content. The project was well appreciated by all our faculty members and we are working on adding a recommendation system so that the user can get suggestions based on his/her previous searches.
